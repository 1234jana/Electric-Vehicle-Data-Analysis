# 🚗 Electric Vehicle Data Analysis

## 📌 Project Overview

This project focuses on analyzing Electric Vehicle (EV) data to uncover meaningful insights and trends. The analysis was performed using **Excel** for data cleaning and preprocessing, and **Tableau** for interactive data visualization and dashboard creation.

The goal of this project is to understand EV adoption patterns, vehicle distribution, and key factors influencing the growth of electric vehicles.

---

## 🛠️ Tools & Technologies Used

* 📊 Excel – Data cleaning, preprocessing, and transformation
* 📈 Tableau – Data visualization and dashboard creation

---

## 📂 Dataset

The dataset contains information about electric vehicles such as:

* Vehicle Type (BEV, PHEV)
* Model and Manufacturer
* Registration Details
* Location Data
* Range and other specifications

---

## 🔍 Steps Performed

### 1️⃣ Data Cleaning (Excel)

* Removed null and duplicate values
* Standardized column names
* Filtered relevant data
* Prepared dataset for analysis

### 2️⃣ Data Analysis

* Identified trends in EV adoption
* Compared different vehicle types (BEV vs PHEV)
* Analyzed distribution across locations

### 3️⃣ Data Visualization (Tableau)

Created an interactive dashboard including:

* EV distribution by type
* Total number of vehicles
* State-wise analysis
* Trend analysis over time

---

## 📊 Dashboard

The Tableau dashboard provides a clear and interactive view of:

* Total Electric Vehicles
* Growth trends
* Category distribution
* Geographic insights

---

## 📈 Key Insights

* Battery Electric Vehicles (BEVs) show higher adoption compared to PHEVs
* EV usage is concentrated in specific regions
* Significant growth trend observed over time

---

## ▶️ How to Use
